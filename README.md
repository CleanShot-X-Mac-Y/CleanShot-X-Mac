## Overview

CleanShot X Professional Suite is a complete visual communication platform that serves professionals with the capture depth, annotation tools, and sharing infrastructure needed for teams that communicate visually as a core part of their workflow. It transforms screenshot taking from an occasional utility into a primary communication channel with the tools and infrastructure to support that role. The application has become the standard screenshot tool for design teams, developer teams, and support organizations that rely on visual communication.

Screen recording with camera overlay captures screen activity with a configurable picture-in-picture webcam view, producing presentation-quality recordings without a separate screen recording application. Recording controls include pause and resume, audio source selection, and the ability to highlight clicks and keystrokes during recording. Quick trimming edits the recording directly after capture, removing false starts and mistakes without importing into a video editor.

OCR text recognition extracts readable text from any captured screenshot, making it possible to copy text from images, non-selectable interfaces, and other visual content that would otherwise need to be retyped manually. The recognition works on any language and handles mixed-language content gracefully. Extracted text is placed on the clipboard immediately, ready to paste into any application.

Team features through CleanShot Cloud provide shared workspaces where team members can access, comment on, and organize shared screenshots in a central location with permission controls and organizational folders. Team branding applies consistent visual identity to shared captures, and analytics show how shared screenshots are being viewed. Integration with Slack, Notion, and other collaboration tools embeds shared captures directly into team communication channels.

CleanShot X Professional Suite is the screenshot platform for teams that have recognized visual communication as a core workflow requirement and need the capture, annotation, and sharing tools that support visual communication at professional scale. Its end-to-end workflow — from capture through annotation to cloud-shared link — removes every manual step from the screenshot communication process.

## Performance & Integration

CleanShot X is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while CleanShot X processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
